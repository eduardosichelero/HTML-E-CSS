Imagem flexivel

Imagens com tamanhos flexiveis, aumente ou diminua seu navegador para obter diferentes versões.

Usando a tag source media definimos o tamanho maximo para que cada imagem seja carregada.
Sendo assim, a imagem P será carregada enquanto a tela tiver no máximo 750 pixels, a imagem M será carregada enquanto a tela tiver no máximo 1050 pixels, após isso a imagem G é carregada para telas com resoluções maiores.
